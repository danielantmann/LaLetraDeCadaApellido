Parte 3

¿Qué es un Pull Request? ¿Cómo podriamos realizar una? ¿Qué utilidad tiene?

Una Pull Request es una función que nos permite pedir aprobación al equipo sobre un cambio antes de juntarlo con la rama principal.

Para poder utilizar un Pull Request debemos acceder a nuestro repositorio remoto en GitHub y entrar en la pestaña de Pull Request, le damos a new, seleccionamos la rama devDani para comparar los cambios con la rama principal y creamos la Pull Request. Nos pedira un titulo para la solicitud y algo de descripción. Una vez la creamos nos queda esperar a que sea aceptada y unir la rama devDani con la rama principal.

Su utilidad se resume en poder revisar y tener un control sobre los cambios del equipo en el desarrollo del proyecto.
